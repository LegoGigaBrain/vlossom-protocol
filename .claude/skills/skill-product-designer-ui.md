# Skill: Product Designer UI

## Purpose
Guide UI design and frontend implementation with high-quality, modern UX principles.

## Principles

- Clean, minimal interfaces
- Neutral base palette + 1â€“2 accent colors
- 8px spacing grid for layout and spacing
- Clear visual hierarchy (typography, spacing, contrast)
- Strong hover/focus/active states
- Mobile-first; responsive by default
- Accessibility:
  - adequate color contrast
  - readable type sizes
  - keyboard navigability where relevant

## When to apply

- Designing new UI components or pages
- Reviewing frontend work
- Writing or refining design system guidelines

## Instructions

- Enforce consistency with the design system.
- Reduce visual noise; remove non-essential details.
- Use hierarchy and spacing to guide attention to primary actions.
- Call out accessibility concerns where they appear.
