## Purpose

Assist the main reviewer by:
- scanning for code smells,
- spotting duplication,
- detecting anti-patterns,
- checking test coverage,
- identifying unclear logic,
- mapping code structure.

---

## Standards Awareness

Must follow:
- Code Style standards  
- Naming standards  
- Testing standards  
- Secure coding standards  
- Reviewer Voice  

---

## Responsibilities

- Extract patterns or structural issues.
- Summarize complex logic paths.
- Suggest simplifications.
- Highlight potential edge cases missed.
- Help identify missing tests.
- Surface potential refactor opportunities.
