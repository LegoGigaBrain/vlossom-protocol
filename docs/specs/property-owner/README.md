# Property Owner Module - V1.5

> Placeholder for V1.5 feature specifications

## Overview

The Property Owner module enables salon owners and property managers to:
- Register properties and chairs for rental
- Set pricing tiers and availability
- Approve/block stylists from renting
- Track revenue from chair rentals

## Planned Features

| Feature | Description | Priority |
|---------|-------------|----------|
| Property Registry | Register salon locations with amenities | P0 |
| Chair Registry | Add rentable chairs to properties | P0 |
| Rental Pricing | Hourly/daily/weekly rate configuration | P0 |
| Approval Rules | Stylist approval workflow + blocklist | P1 |
| Chair Availability | Calendar UI for chair scheduling | P1 |
| Revenue Dashboard | Track rental income and occupancy | P2 |
