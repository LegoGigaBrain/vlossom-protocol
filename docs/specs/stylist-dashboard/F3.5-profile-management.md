# F3.5: Profile Management

**Status**: Pending
**Priority**: Medium (Week 6)

## Purpose

Allow stylists to manage their professional profile displayed to customers, including bio, location, operating mode, and portfolio images.

## User Story

As a stylist, I want to create an attractive professional profile, so customers can learn about me and trust my services.

## Form Validation

| Field | Rules |
|-------|-------|
| Display Name | Required, 2-50 characters |
| Bio | Required, 50-500 characters |
| Operating Mode | Required, enum: FIXED, MOBILE, HYBRID |
| Base Location | Required for FIXED/HYBRID |
| Service Radius | Required for MOBILE/HYBRID, 5-100 km |
| Specialties | Optional, max 10 items |
| Portfolio | Optional, max 12 images, max 5MB each |

## Operating Modes

| Mode | Description | Location Required | Radius Required |
|------|-------------|-------------------|-----------------|
| FIXED | Work from one location | Yes | No |
| MOBILE | Travel to customers | Optional | Yes |
| HYBRID | Both options | Yes | Yes |

## Image Upload

### Supported Formats
- JPEG, PNG, WebP
- Max file size: 5MB
- Recommended: 800x800px minimum

### Storage
- Images uploaded to Cloudinary
- Automatic optimization and CDN delivery
- Thumbnails generated automatically

- [ ] Can use current location for base address
- [ ] Can add/remove specialties
- [ ] Can upload portfolio images
- [ ] Can delete portfolio images
- [ ] Can reorder portfolio images (drag & drop)
- [ ] Preview shows customer view accurately
- [ ] Validation errors shown inline
