## Auto-Confirm Job (Backend)

If customer doesn't confirm within 24 hours:
1. Job runs hourly, checks for AWAITING_CUSTOMER_CONFIRMATION bookings
2. If `completedAt + 24h < now()`, auto-confirm
3. Release funds from escrow to stylist wallet
4. Mark as SETTLED

## Acceptance Criteria

- [ ] "Start Service" button only visible for CONFIRMED bookings

- [ ] Elapsed time displays while in progress
- [ ] "Complete Service" button visible for IN_PROGRESS bookings
- [ ] Completing shows payout breakdown
- [ ] Completion changes status to AWAITING_CUSTOMER_CONFIRMATION
- [ ] Success modal shows expected payout
- [ ] Actual duration recorded and displayed
- [ ] Loading states during API calls
- [ ] Error handling with retry option
