# Vlossom Beta Program

Welcome to the Vlossom beta program! Thank you for being an early user and helping us shape the future of the natural hair salon economy.

## What is Vlossom?

Vlossom is a marketplace connecting customers with hair stylists, powered by blockchain technology for secure, transparent payments.

### Key Features

- **Smart Wallets**: Automatic wallet creation with no crypto knowledge required
- **Escrow Payments**: Funds held securely until service completion
- **Gas-Free Transactions**: We cover all blockchain transaction fees
- **Fair Pricing**: Transparent platform fees with no hidden costs

## Beta Program Details

### What to Expect

- **Test Environment**: All transactions use test USDC on Base Sepolia (testnet)
- **Free Test Funds**: Claim 1,000 test USDC from our faucet
- **Real Features**: Full functionality to experience the platform
- **Feedback Welcome**: Help us improve by reporting issues

### Known Limitations

During beta, some features are limited:

1. **Testnet Only**: Real money transactions not yet available
2. **Limited Region**: Currently testing in South Africa
3. **Mobile App**: Web-only during beta (mobile coming soon)
4. **Payment Methods**: Faucet only (MoonPay integration coming soon)

## Quick Start

### For Customers

1. **Sign Up**: Create account at [vlossom.io](https://vlossom.io/signup)
2. **Claim Funds**: Get test USDC from the faucet in your wallet
3. **Browse Stylists**: Find stylists in your area
4. **Book Service**: Select a service and time slot
5. **Complete Booking**: Funds held in escrow until service completion

### For Stylists

1. **Sign Up**: Create account and select "Stylist" role
2. **Complete Profile**: Add bio, photos, and location
3. **Add Services**: Define your offerings with prices
4. **Set Availability**: Configure your working hours
5. **Accept Bookings**: Receive and approve customer requests
