# LEGO OS Commands

Slash commands orchestrate multi-agent workflows.  
Use them to perform complex engineering tasks consistently.

---

# Review Commands

## `/pragmatic-code-review`
Use for:
- reviewing PRs  
- evaluating generated code  
- checking logic correctness  
- validating test coverage  

---

## `/design-review`
Use for:
- reviewing UI screens  
- reviewing flows  
- checking hierarchy, clarity, spacing  
- validating design system usage  

---

## `/ux-review`
Use for:
- evaluating task flows  
- identifying friction or confusion  
- improving onboarding  

---

## `/security-review`
Use for:
- sensitive logic  
- auth  
- payments  
- smart contract interactions  

---

## `/smart-contract-review`
Use for:
- token contracts  
- escrow pools  
- liquidity mechanics  
- role permissions  
- invariant checks  

---

## `/architecture-review`
Use for:
- microservice boundaries  
- event flows  
- system decomposition  
- API design  
- scalability concerns  

---

## `/spec-review`
Use for:
- validating feature specifications  
- checking completeness & clarity  
- ensuring feasibility  
- aligning engineering with product intent  

---

## `/migration-review`
Use for:
- database schema changes  
- backfills  
- type changes  
- index additions  
- data migrations at scale
